# AI Pertanian Pandas

Project analisis data produksi padi menggunakan Python dan Pandas.

## Fitur
- Membaca data CSV
- Menghitung total produksi padi
- Filter data berdasarkan tahun
- Menyimpan laporan hasil analisis

## Struktur Folder
data/ -> dataset mentah
output/ -> hasil laporan
script/ -> module analisis
main.py -> program utama

## Teknologi
- Python
- Pandas
- Git & GitHub
